## ğŸš€ Getting Started

### Prerequisites
- Node.js (v16+)
- npm or yarn
- A code editor (VS Code recommended)
- Git

### Installation

```bash
# Clone the repository
git clone https://github.com/Sahil-Singh23/cohort.git
cd COHORT

# Install dependencies
npm install
```

### Running Examples

```bash
# TypeScript
cd TypeScript
npx ts-node index.ts

# Node.js projects
cd projects/auth-class
npm install
npm start

# React projects
cd REACT/first-react-app
npm start
```
